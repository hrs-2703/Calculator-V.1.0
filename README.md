# Calculator-V.1.0

#  Python Calculator App

A simple Flask-based calculator application with CI/CD using GitHub Actions, unit testing, linting, code formatting, code coverage reporting, and Docker containerization.

---

##  Features

- REST API for basic arithmetic operations: add, subtract, multiply, divide
- Built with Flask (Python)
- Unit testing with `pytest`
- Code linting with `flake8`
- Code formatting with `black`
- Code coverage reporting with `pytest-cov` and Codecov
- CI/CD pipeline via GitHub Actions
- Dockerized with Dockerfile + docker-compose

source venv/bin/activate   # macOS/Linux
venv\Scripts\activate      # Windows
